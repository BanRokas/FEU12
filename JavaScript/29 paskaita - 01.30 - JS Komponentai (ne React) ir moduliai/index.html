Moduliai

Moduliai yra atskiri kodo fragmentai išskaidyti skirtinguose failuose.

Moduliuose gali būti įvairus kodas. Nuo duomenų (kintamųjų) iki funkcijų, klasių, metodų ir kt.
JS Klasės turi būti atskiri failai. Kitaip pasakius - viename faile neturėtų būti daugiau negu viena JS Klasė.
HTML'e prijungas skriptas, kuris kreipiasi į kitus script failus, PRIVALO turėti nurodytą tipą - modulis. (type='module')
HTML failas, kuriame yra modulio tipo JS failas, PRIVALO būti paleistas kaip serveris, kitu atveju neleis vykdyti JS failo kodo.

Export ir Import

Failuose esanti informacija gali būti iškeliama (export'uojama) ir galima kažkokią informaciją įsikelti (import'uoti) iš kito failo.

Export

Informaciją galima eksportuoti dvejais būdais:

  • export default - numatytasis/pagrindinis export'as. Jis gali būti TIK VIENAS viename faile.
  • export - paprastas/papildomas export'as/ai, kurių gali būti daug viename faile. Jie visi gali būti išrašyti vienoje eilutėje.

Import

Informacija gali būti importuojama įvairiais būdais.
Bet yra keletas pagrindinių įdėjų:

  • import RandomDefaultName from './.../*.js' - kai yra importuojamas default export'as galima jo pavadinimą pasirašyti betkokį.
  • import { sameName as diffName } from './.../*.js' - yra importuojamas paprastas export'as kitokiu pavadinimu.
  • import DefaultStuff, { nonDefaultStuff } from './.../*.js' - viename import aprašyme galima importuoti ir default ir paprastus export'us.
