Moduliai

Moduliai yra atskiri kodo fragmentai išskaidyti skirtinguose failuose.

Moduliuose gali būti įvairus kodas. Nuo duomenų (kintamųjų) iki funkcijų, klasių, metodų ir kt.
JS Klasės turi būti atskiri failai. Kitaip pasakius - viename faile neturėtų būti daugiau negu viena JS Klasė.
HTML'e prijungas skriptas, kuris kreipiasi į kitus script failus, privalo turėti nurodytą tipą - modulis. (type='module')
HTML failas, kuriame yra modulio tipo JS failas, privalo būti paleistas kaip serveris, kitu atveju neleis vykdyti JS failo kodo.

Export ir Import

Failuose esanti informacija gali būti iškeliama (export'uojama) ir galima kažkokią informaciją įsikelti (import'uoti) iš kito failo.
