JSON server

JSON serveris yra NPM (Node Package Manager) paketas, kurio pagalba galima susikurti lokalų API serverį su savo susikurtais duomenimis, kurie privalo būti JSON faile, JSON formatu. Į šį serverį galima bus siūsti fetch užklausas su įvairiais metodais.

Tam, kad galėtume naudotis JSON serveriu mums reikia:

  1. npm (įsirašo kartu su Node'u)
  2. įsirašyti json-server globaliai
  3. susikurti JSON failą tinkama struktūra
  4. pasileisti JSON failą su json-server komanda

Terminalo komandos

  • cd - change directory - vaikščioti tarp aplankalų
  • ls - list files - išvardina failus/aplankalus esamoje direktorijoje (mac/linux)
  • dir - list files - išvardina failus/aplankalus esamoje direktorijoje (windows)
  • json-server fileName.json
  • clear - išvalo komandinės eilutės istoriją (not rlly, tiesiog nukelia aukštyn)
  • CTRL C - atšaukia dabartinę komandą/veiksmą/eilutę...
  • ARROW UP - paskutinė paleista komanda terminale

Navigavimasis (cd +?)

  • / - projekto/sistemos pradžia
  • ./ - niekur (toje pačioje vietoje lieki)
  • ../ - žingsnis per aplankalus atgal

JSON server Routes

*kiekvienas duomenų laukas (dName) gauna savo route'us*

  • GET dName - gauti visus dName lauke esančius duomenis
  • GET dName/id - gauti dName lauke esantį duomenį su nurodytu ID
  • POST dName - papildyti dName lauke esančius duomenis nauju duomenimi
  • PUT dName/id - redaguoti specifinį duomenį (pagal id) dName lauke viską (apart id) perrašydamas nauja info
  • PATCH dName/id - redaguoti specifinį duomenį (pagal id) dName lauke tą informacijos dalį, kuri buvo perduota
  • DELETE dName/id - trinamas specifinis duomuo (pagal id) dName lauke
