Kintamieji

Kintamųjų tipai

  • String - tekstas
  • Number - skaičius
  • Boolean - loginis
  • Array - masyvas - konteineris kur eiliškumas svarbu ir elementai turi indeksus
  • Object - objektas - konteineris kur eiliškumas nesvarbu, nes elementai turi pavadinimus (raktinius žodžius)

Sąlygos

Sąlygų tipai

  • if elseIf elseIf... else
  • switch
  • ternary
  • logic AND

Sąlygos yra reikalingos - jeigu sąlyga yra teisinga tuomet vykdytų jos viduje aprašytą kodą.

Ciklai

Ciklų tipai

  • for(skaitliukas; sąlygaSuSkaitliuku; skaitliukoKitimas){ veiksmai }
  • for(el of array){ veiksmai }
  • for(key in obj){ veiksmai }
  • while(sąlyga){ veiksmai }
  • do{ veiksmai }while(sąlyga)
  • iteraciniai masyvų ciklai

Ciklai yra reikalingi - vykdyti kažkokius vienodus ar panašius veiksmus kažkokį kiekį kartų.

Funkcijos

Funkcija yra reikalinga - vykdyti joje aprašytą kodą tada, kada tą funkciją iškviečiame. Funkciją galime iškviesti daug kartų. Funkciją galime iškviesti perduodami skirtingus argumentus (duomenis).

Parametras - duomuo, kuris yra aprašytas funkcijos kūrimo metu. Šis duomuo naudojamas funkcijos viduje ir priklausomai nuo iškvietimo gali kiekvieną sykį skirtis.

Argumentas - duomuo, kuris yra perduodamas funkcijai jos iškvietimo metu. Skirtingų iškvietimų metu tas duomuo gali būti vis kitoks.

Funkcija gali grąžinti kažkokią reikšmę.
