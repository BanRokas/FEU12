Kintamieji

Kintamųjų tipai

  • String - tekstas
  • Number - skaičius
  • Boolean - loginis
  • Array - masyvas - konteineris kur eiliškumas svarbu ir elementai turi indeksus
  • Object - objektas - konteineris kur eiliškumas nesvarbu, nes elementai turi pavadinimus (raktinius žodžius)

Sąlygos

Sąlygų tipai

  • if elseIf elseIf... else
  • switch
  • ternary
  • logic AND

Sąlygos yra reikalingos - jeigu sąlyga yra teisinga tuomet vykdytų jos viduje aprašytą kodą.

Ciklai

Ciklų tipai

  • for(skaitliukas; sąlygaSuSkaitliuku; skaitliukoKitimas){ veiksmai }
  • for(el of array){ veiksmai }
  • for(key in obj){ veiksmai }
  • while(sąlyga){ veiksmai }
  • do{ veiksmai }while(sąlyga)
  • iteraciniai masyvų ciklai
