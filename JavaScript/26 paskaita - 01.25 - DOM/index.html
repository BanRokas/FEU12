HTML formų validavimas

Formų validavimas - tikrinimas ar formos įvesčių laukai teisingai supildyti.
Formų validavimas yra reikalingas tam, kad nebūtu leidžiama pateikti formos, jeigu kažkurie laukai neteisingai supildyti arba išvis nesupildyti.

  • required - įvesties laukas privalomas
  • minlength - minimalus kiekis simbolių - text/password/url/email
  • maxlength - maksimalus kiekis simbolių - text/password/url/email
  • min - mažiausias galimas skaičius - number/range/date
  • max - didžiausias galimas skaičius - number/range/date
  • step - kokiu žingsniu galima įvesti skaičių - number/range/date
  • pattern - spacifikuota seka (su regex) - password/tel
  • accept - kokio tipo failą/us priima - file
  • multiple - leidimas daugiau įvesčių į vieną - file/email
