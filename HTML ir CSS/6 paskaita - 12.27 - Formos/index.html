Formos

Formos tag'ai

  • form - forma
  • label - etiketė įvesties laukui
  • input - įvairus įvesties laukas
  • textarea - teksto (didelio kiekio) įvesties laukas
  • select - pasirinkimo variantai (mažai)
  • datalist - pasirinkimo variantai (daug)
  • option - pasirinkimo variantas

Formos atributai

  • for - etiketės atributas skirtas ją sujungti su įvestimi
  • id - label sujungimui su įvestimi (turi būti toks pats tekstas kaip ir for)
  • name - priskiria įvesčiai vardą (dažniausiai toks pats kaip ir id)
  • type - skirtas nurodyti input'o paskirtį
    • text - paprastas tekstas
    • password - slaptažodis (slepiamas tekstas)
    • email - elektroninis paštas
    • number - skaičius
    • url - nuoroda
    • file - failas iš įrenginio
    • checkbox - multi variantų pasirinkimas (privalo turėti vienodas name atributų reikšmes ir skirtingas value atributų reikšmes)
    • radio - vieno iš keletos variantų pasirinkimas (privalo turėti vienodas name atributų reikšmes ir skirtingas value atributų reikšmes)
    • date - datos pasirinkimas (metų, mėnesio, dienos)
    • datetime-local - datos ir laiko pasirinkimas (metų, mėnesio, dienos, valandų ir minučių)
    • month - mėnesio pasirinkimas
    • week - savaitės pasirinkimas
    • time - laiko pasirinkimas
    • color - spalvos pasirinkimas
    • button - mygtukas be funkcionalumo
    • submit - formos pateikimo input'as mygtuko pavidalu
    • image - formos pateikimo input'as paveiksliuko pavidalu
    • hidden - paslepia įvestį, bet reikšmė yra pateikiama
    • range - slenkančios juostos skaičiaus pasirinkimas
    • reset - mygtukas išvalantis formos įvestis
    • search - paieškos laukas
    • tel - telefono įvesties laukas
  • required - privalomas įvesties laukas (be jo neleis pateikti formos)
  • min - minimali reikšmė (skaičiaus)
  • max - maksimali reikšmė (skaičiaus)
  • step - žingsnis (skaičiaus)
  • type - failo tipas ar tipai, kuriuos leidžiama pateikti
  • pattern - kokia seka kažkas turi būti parašyta (regex :))
  • placeholder - tekstas atvaizduojamas input'o arba textarea viduje
  • checked - iš anksto pažymėtas radio arba checkbox
  • cols - textarea dydžio nustatymai
  • rows - textarea dydžio nustatymai
  • value - įvesties reikšmė
  • selected - iš anksto parinktas option'as
  • disabled - neleidžiamas pasirinkti option'as
