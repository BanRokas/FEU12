Paveldimumas

Elementai paveldi stilius iš tėvinio elemento.

Elementai perduoda stilius savo vaikiniams elementams.

Naršyklės numatytieji stiliai

Kiekvienas (beveik) tag'as turi savo numatytuosius naršyklės stilius, kurie pakeičia jo vaizdą.
Skirtingos naršyklės gali turėti skirtingus numatytuosius stilius tiems patiems tag'ams.
Tam, kad tag'ų stiliai būtų vienodi arba būtų panaikinami nenaudingi naršyklių stiliai, yra naudojami Reset arba Normalize CSS'ai.

Reset CSS - suvienodina visų tag'ų stilius per visas naršykles. (asmeniškai nesiūlau naudoti)

Normalize CSS - suvienodina vienodų tag'ų stilius tarp skirtingų naršyklių ir panaikina nenaudingus naršyklių stilius. (asmeniškai siūlau naudoti)

Nuoroda

Esant dviems selektoriams, kurie selektina tą patį elementą, turi vienodą prioritetą ir keičia tą patį stilių yra atsižvelgiama į tų selektorių eiliškumą. Vėliau aprašyto tokio selektoriaus stiliai turi prioritetą.

Jeigu įmanoma - nenaudoti!!! !important prierašo. !important ignoruoja visas specifikacijas, visus eiliškumus, visus prioritetus ir suteikia nurodytą stilių (stiliausVardas: stiliausNustatymas !important;). Vienintelis būdas perrašyti !important yra naudojant kitą !important...
