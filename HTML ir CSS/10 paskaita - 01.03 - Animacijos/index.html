Transformacija

Transformacija nurodo kaip kinta elementas. Elementas gali kisti 3-5 būdais.

  • translate - judina elementą nurodytoje ašyje
    • translate(x, y)
    • translate3d(x, y, z)
    • translateX(x)
    • translateY(y)
    • translateZ(z)
  • rotate - rotuoja
    • rotate(z)
    • rotate3d(x, y, z, a)
    • rotateX(x)
    • rotateY(y)
    • rotateZ(z)
  • scale - keičia dydį / tempia/spaudžia
    • scale(x, y)
    • scale3d(x, y, z)
    • scaleX(x)
    • scaleY(y)
    • scaleZ(z)
  • skew - tampo kampus
    • skew(x, y)
    • skewX(x)
    • skewY(y)
  • matrix - universalus
  • matrix3d - universalus3d
  • perspective - perspektyvos keitimas

Kartu su transform gali būti naudojamas ir transform-origin, kuris nusako transformacijos pradinį tašką. Default yra centras.

Tranzicija

  • transition-duration - per kiek laiko turi įvykti pasikeitimas
  • transition-timing-function - kaip pasikeitimo laikas yra paskirstytas (ease)
  • transition-property - nurodo kuris stilius turi pasikeisti (all)
  • transition-delay - po kiek laiko prasidės kitimas
  • transition - visų junginys (duration, delay (kiti nesvarbu))

Animacija

Animacijos nustatymai

  • animation-name - animacijai suteiktas vardas
  • animation-duration - animacijos trukmė
  • animation-iteration-count - kiek sykių vyks animacija (default 1)
  • animation-play-state - animacijos būsena (vykdoma ar ne)
  • animation-timing-function - kaip pasikeitimo laikas yra paskirstytas (ease)
  • animation-direction - kryptis
  • animation-delay - animacijos uždelsimas
  • animation-fill-mode - ką daryti prieš/po animacijos
  • animation-composition - ką daryti su stiliais kurie turi kelis nustatymus
  • animation - apjungimas (duration, delay, ...)

Animacijos aprašymas (veiksmai / žingsniai)

Animacjos žingsniai aprašomi naudojant keyframes. Keyframes gali būti nurodomi specifiniais žodžiais arba procentais. Nuo 0 iki 100. From -> to. Kiek procentų žingsniui yra skiriama - tiek apytiksliai laiko iš duration'o ir bus suteikiama tam žingsniui atlikti.

        @keyframes animation-name{
          0%{
          } x% {
          } y% {
          } ...% {
          } 100% {
          }
        }
