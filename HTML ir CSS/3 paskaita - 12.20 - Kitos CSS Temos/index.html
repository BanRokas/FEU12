Semantika

Puslapio semantika yra reikalinga norint turėti tvarkingą puslapio struktūrą, kas pagelbės tiek susigaudant puslapio išdėstyme tiek esant reikalui stilizuoti atitinkamus elementus. Semantiškai tvarkingas puslapis turi aukštesnį google paieškos algoritmo balą, kuris padeda vartotojams surasti jūsų puslapį.

  • header - puslapio pagrindinis menu (viršuje)
  • nav - navigacija
  • main - puslapio info atvaizdavimo dalis
  • section - puslapio skiltis
  • article - puslapio skiltis
  • aside - šoninė juosta
  • footer - puslapio apačia
  • figure - paveiksliuko apipavidalinimas
  • figcaption - paveiksliuko pavadinimas
