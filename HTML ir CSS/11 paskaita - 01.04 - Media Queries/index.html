Media Queries

Media Queries yra naudojami norint svetainei suteikti skirtingus stilius esant skirtingiems ekranų dydžiams.
CSS'e yra rašoma @media (min arba max-width: ...px){ ribos kur rašomas CSS tam ekrano dydžiui }

Media Queries ekrano lūžio taškai (breakpoints)

  • no media mažesniam negu 576px - telefonai portretiniai
  • didesniam negu 576px - telefonai peizažiniai
  • didesniam negu 768px - plančetės
  • didesniam negu 992px - maži desktop (laptop)
  • didesniam negu 1200px - desktop
  • didesniam negu 1400px - large desktop

No media, kodėl?

Pradiniai stiliai turi būti rašomi be jokio media querie. Nes media querie dažniausiai pakeičia tik elementų išdėstymą arba atvaizdavimą/neatvaizdavimą, bet ne kažkokius bendrus stilius (fono spalvą, rėmelius, teksto spalvą, šriftą...). Bendri stiliai yra taikomi visiems ekranų dydžiams, tai kad jų nekartoti kiekviename media query - juos aprašai be media querie ir tuomet jie taikomi visiems ekranų dydžiams.
