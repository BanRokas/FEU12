CSS rašymo būdai

  • Inline
  • Internal
  • External

Inline

Inline yra CSS rašymo būdas, kai stilius yra suteikiamas elementui jį aprašant style atributo viduje.

Šitas CSS rašymo būdas naudojamas norint kažką pasitikrinti arba esant reikalui kažką perrašyti nepaisant prioritetų.

Lorem ipsum dolor sit amet.

Internal

Internal yra CSS rašymo būdas, kai stiliai yra aprašomi tame pačiame HTML faile, style tag'o viduje, kuris talpinamas head tag'o viduje. Sintaksė tokia pati kaip External CSS'o.

Šitas CSS rašymo būdas yra naudojamas norint kažką pasitikrinti arba esant atskiram mažam HTML'o failui, kuriam reikalingas unikalus CSS'as.

Lorem ipsum dolor sit amet.

External

External yra CSS rašymo būdas, kai stiliai yra aprašomi atskirame CSS faile, kuris yra prijungiamas prie HTML'o.

Šitas CSS būdas yra naudojamas dažniausiai. Jo didžiausi pliusai yra perpanaudojimas ir tvarkingumas.

CSS Selektoriai

  • * - visko selektorius
  • tagName - tag'o selektorius
  • .className - klasės selektorius
  • #idName - id selektorius

* - parenka visiškai visus elementus

tagName - parenka visus elementus su nurodytu tag'o pavadinimu

Klasė

.className - parenka visus elementus, kurie turi klasės atributą su tokiu pačiu vardu.

Viena klasė, gali būti priskirta daugeliui elementų.

Vienas elementas, gali turėti daugelį klasių.

ID

#idName - parenka visus elementus, kurie turi id atributą su tokiu pačiu vardu.

Vienas ID, gali būti priskirtas TIK VIENAM elementui.

Vienas elementas, gali turėti TIK VIENĄ ID.

Pavadinimų kūrimas

  • camelCase - kiekvienas naujas žodis prasideda iš didžiosios raidės
  • snake_case - kiekvienas naujas žodis prasideda už apatinio brūkšnelio
  • kebab-case - kiekvienas naujas žodis prasideda už brūkšnelio per vidurį

Colors

  • name
  • rgb
  • hex
  • hsl

Name - nurodomas spalvos vardas. Spalvų vardų yra 140.

RGB - Red Green Blue

rgb(red, green, blue) spalvos kiekis nurodomas nuo 0 iki 255

rgba(red, green, blue, alpha) red,green,blue nuo 0 iki 255; o alpha nuo 0 iki 1, 0.001 žingsniu - permatomumas

Maksimalus spalvų kiekis neskaitant alpha yra 256*256*256=16777216 spalvų.

Hex - hexdecimal - šešioliktainis

hex spalvos kiekis nurodomas nuo 0 iki f

Inline - Linijinis

Linijinis elementas užima tik tiek vietos kiek reikia jo viduje esančiam turiniui atvaizduoti. Keletas linijinių elementų gali būti išsidėstę vienas šalia kito.

  • a
  • span
  • input
  • label
  • textarea
  • ...

Linijiniai elementai negali turėti viršutinio ir apationio margin'o (atitraukimas nuo aplinkinių elementų). Linijiniai elementai negali turėti pločio ir aukščio nustatymų.

Block - Blokinis

Blokinis elementas užima visą galimą plotį. Tai reiškia, kad keletas blokinių elementų negali būti išsidėstę vienas šalia kito.

  • h1-h6
  • p
  • div
  • ol/ul
  • li
  • form
  • table
  • ...

Blokiniai elementai gali turėti margin'ą ir pločio bei aukščio nustatymus.

Positions

  • static
  • relative
  • absolute
  • fixed
  • sticky

Suteikus elementui betkurią apart static pozicijos, jis įgauna galimybę naudotis šiais stiliais: z-index, top, right, bottom, left;

z-index nusako elemento eilės sluoksnį ekrane (kaip arti / toli nuo vartotojo yra elementas (katras elementas katrą užstos))

Static

Static nustatymas nesuteikia jokių ypatybių. Net galimybės naudotis: z-index, top, right, bottom, left;

Relative

Relative nustatymas suteikia elementui galimybę naudotis z-index, top, right, bottom, left nustatymais.
Top, right, bottom, left nustatymai pastumia elementą nuo nurodytos kraštinės per nurodytą atstumą nuo elemento originalios pozicijos.
Elemento viduje esantis absolute pozicijos elementas, atsitraukia nuo elemento kraštų jei šis turi relative poziciją.

Absolute

Absolute nustatymas suteikia elementui galimybę naudotis z-index, top, right, bottom, left nustatymais.
Top, right, bottom, left nustatymai nurodo per kokį atstumą elementas atsitrauks nuo artimiausio protėvinio elemento, turinčio relative poziciją, ribų.
Elementas turintis absolute pozicija išeina iš puslapio tėkmės. (jo pozicijos neįtakoja puslapyje išsidėstę elementai)

Fixed

Fixed nustatymas suteikia elementui galimybę naudotis z-index, top, right, bottom, left nustatymais.

Elementas su fixed pozicija išeina iš puslapio tėkmės.
Fixed elementas užsifiksuoja vienoje ekrano vietoje ir kad ir kiek būtų scroll'inama, jis juda kartu.
Top, right, bottom, left nustatymai nurodo per kokį atstumą, nuo kurios viewport kraštinės jis atsitraukia ir užsifiksuoja.

Sticky

Fixed nustatymas suteikia elementui galimybę naudotis z-index, top, right, bottom, left nustatymais.

Sticky poziciją turintis elementas prilimpa prie ekrano (viewport) kraštų per nurodytą atstumą ir atkimba tik pasiekęs tėvinio elemento ribas arba grįžęs į savo originalią poziciją.
