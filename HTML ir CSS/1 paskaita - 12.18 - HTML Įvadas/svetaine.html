  • src - source - failo nuoroda
  • defer - nurodymas, kad failą skaitytų/vykdytų po viso HTML perskaitymo
  • style - inline stilizavimui skirtas atributas
  • class - klasės suteikimas
  • id - id suteikimas
  • width - pločio nustatymas
  • height - aukščio nustatymas

  • title - atvaizduojamas tekstas užvedus pelę ant elemento
  • type - formos input'o tipo nurodymas
  • for - label sujungimas su input/textarea/select/fieldset
  • name - input/textarea/select/fieldset įvesties lauko pavadinimas

  • colspan - nurodo kelis stulpelius td arba th turi užimti
